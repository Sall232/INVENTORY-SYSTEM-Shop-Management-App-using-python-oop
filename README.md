# INVENTORY SYSTEM (Shop Management App)

## Overview

INVENTORY SYSTEM (Shop Management App) is a Python-based inventory management application developed using Object-Oriented Programming (OOP) principles. The system is designed to help shop owners and small businesses manage products, track stock levels, process sales, and generate inventory reports.

The project serves as a practical implementation of core Python OOP concepts including classes, objects, methods, encapsulation, and modular programming.

## Features

### Product Management

* Create and manage products.
* Store product information including name, price, and quantity.

### Stock Tracking

* Monitor inventory levels.
* Add new stock to existing products.
* Prevent negative stock quantities.

### Sales Processing

* Sell products directly from inventory.
* Automatically update stock quantities after each sale.
* Validate stock availability before completing a sale.

### Inventory Reporting

* Calculate the current total value of inventory.
* Generate inventory summaries for business monitoring.
